## рџ“Њ Overview

This repository contains the official course deliverables for the **AI/LLM Inventory Management System**, a capstone project for **UMGC CMSC 495**.

The project is designed to help restaurants and small food businesses reduce food waste, improve inventory tracking, support menu planning, and make better operational decisions through structured inventory data and AI-assisted recommendations.

This repository is focused on **course documentation and submission materials**. Application source code should be maintained in the appropriate frontend, backend, or full-stack development repository unless required for course submission.
